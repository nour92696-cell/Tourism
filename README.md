# Explorer le Tourism en Tunisie
##  Description
Ce projet est une plateforme interactive (ou application) dédiée à la promotion du tourisme en Tunisie. 
L'objectif est de mettre en avant la richesse culturelle, historique et naturelle du pays à travers 
une interface moderne. Le projet permet aux utilisateurs de découvrir les destinations incontournables 
(Sidi Bou Saïd, Carthage, Sahara, etc.), de consulter des guides locaux et d'explorer les activités disponibles.
##  Fonctionnalites (Features)
-  **Assistant Intelligent (Chatbot):** Un chatbot integré (basé sur OpenAI) pour répondre aux questions des touristes.Aide à l'organisation des séjours.Aide à la Découverte des sites touristiques par catégorie (Plages, Histoire, Désert)etc...
## Technologies utilisées
- **Frontend:** HTML5, CSS3, JavaScript , Python
- **Version Control:** Git & GitHub
- **Intelligence Artificielle:** API OpenAI / Chatbot
